Minesweeper

Rules

To win, you must uncover all safe squares without clicking on a mine.

The game board is a grid of squares. Some squares contain hidden mines, and the rest are safe. You can click (or tap) a square to reveal whatâ€™s underneath. If the square contains a mine, the game is over. If the square is safe, it will display a number or be blank. The number shows how many mines are in the surrounding eight tiles.

Controls (desktop only)

  • Short ClickReveals a square
  • Long ClickPlaces a flag
